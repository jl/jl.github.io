extract doc bookmarklet

This bookmarklet will extract document page images into a new tab that can be printed (or saved to PDF).

  1. Add the following link as a bookmark (right click on it, add bookmark).
  2. Go back to the tab with the document you want to extract.
  3. Make sure the document is open, then activate the "extract doc" bookmark you just created.
  4. This should open a new tab, which (after a brief delay) you can read, print, or save to PDF.

Here is the bookmarklet:

extract doc

Note: When printing or saving to PDF, select "Fit to page width" for best results.
